# AI-Powered Gmail Email Assistant

## Overview
This project automates Gmail email responses using n8n workflow automation and Google Gemini AI.

---

## Workflow
- Gmail Trigger detects new incoming emails
- Email content is extracted
- Gemini AI generates professional reply
- Gmail sends automated response

---

## Tech Stack
- n8n
- Gmail API
- Google Gemini AI
